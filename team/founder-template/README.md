> **This is example content shipped with project-kb.** Copy this folder to `team/<your-slug>/` and replace the content with your own profile. Delete this template once you have your own team folder.

## Role

Founder. Responsible for product direction, fundraising, and customer development.

## Current focus

What you're owning this quarter. Updated as it changes — this section is **current truth, not a timeline**. The folder's dated sibling files form the timeline.

## Background

One paragraph on your background — what you've built before, what domain expertise you bring. Kept short and current.

## Working preferences

- Working hours and timezone
- Async-vs-sync default (e.g., "default to async; sync calls scheduled via Calendly")
- Communication channels (Slack, email, etc.)
- Any explicit "don't ping me about X" or "always ping me about Y"

If using the optional voice plugin (`plugins/voice/`), drop your voice corpus in `team/<your-slug>/voice/` (style-card.md + reverse-prompts.md). See [[plugins/voice/README]] for the install.

## Private content

Per [`CONVENTIONS.md`](../../CONVENTIONS.md), per-team-member private files (resumes, gov-ID docs, anything not for the repo) live under `team/<your-slug>/private/` which is gitignored. Agents grepping the team profile can still read those files locally, but they never reach the remote.

## Interactions

Dated one-liners auto-appended by `/complete-todo` when a todo with `related: [[team/<your-slug>]]` is marked complete. Most recent at the bottom.
